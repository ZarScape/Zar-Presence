### 🚀 About ZarPresence
ZarPresence is a modern Discord Rich Presence solution that automatically syncs your activity with privacy and performance in mind.

- **Fast & Lightweight**: Built with React and Vite.
- **Dynamic Updates**: Fetches real-time version info from GitHub.
- **Modern UI**: Dark-themed design with Framer Motion animations.
